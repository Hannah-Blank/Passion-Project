# 🏎️ F1 Strategy Predictor

> **Status: Work in Progress** — Currently in the learning phase. Active development begins Summer 2026.

A machine learning web app that analyses Formula 1 race strategy using real telemetry data. Users can explore historical pit stop strategies, compare drivers, and simulate "what if" scenarios — for example: *"What if Lando Norris had pitted one lap earlier at Monaco?"*

---

## 🎯 What This Project Does

Formula 1 race strategy is one of the most complex and fascinating aspects of the sport. Decisions made in seconds — when to pit, which tyre to switch to, whether to cover an opponent — can determine the outcome of an entire race. This project aims to make those decisions transparent and explorable using real data and machine learning.

**The app will allow users to:**
- Select any race from the 2023 or 2024 F1 season
- Visualise each driver's tyre strategy across the full race
- Compare lap time performance between drivers
- Simulate alternative pit stop strategies and see predicted finishing positions
- Explore how small strategic decisions can change race outcomes
